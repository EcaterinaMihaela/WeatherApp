OpenWeatherMap One Call API sau 5-day/3-hour forecast API.

1️⃣ Tipuri de API OpenWeatherMap

Current Weather Data – ce folosești acum, dă doar vremea curentă.

5 Day / 3 Hour Forecast – dă prognoza pe 3 ore pentru următoarele 5 zile.

URL:

http://api.openweathermap.org/data/2.5/forecast?q={city}&appid={API_KEY}&units=metric


Primești un JSON cu list – fiecare element are:

dt_txt – data și ora

main.temp – temperatura

weather[0].description – condiție meteo

pop – probability of precipitation (%)

One Call API (mai avansat) – dă current, hourly, daily, alerts.

URL:

https://api.openweathermap.org/data/3.0/onecall?lat={lat}&lon={lon}&exclude=minutely,alerts&appid={API_KEY}&units=metric


Primești:

current – vreme curentă

hourly – prognoza pe ore

daily – prognoza pe zile

Important: One Call API necesită latitude și longitude, deci mai întâi trebuie să faci un call pentru city -> coordonate.

2️⃣ Cum să modifici WeatherService

Creezi o nouă metodă care să apeleze 5-day/3-hour forecast sau One Call API.

Extragerea datelor din JSON poate fi făcută folosind JSONObject sau Map.
